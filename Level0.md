## Level 0

### :trophy: Level Goal

The goal of this level is for you to log into the game using SSH. The host to which you need to connect is bandit.labs.overthewire.org, on port 2220. The username is bandit0 and the password is bandit0. Once logged in, go to the Level 1 page to find out how to beat Level 1.

```
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
One you write the connection down you'll be connected to OverTHeWire game. The password is the one they provided so we wont be covering that here.

:exclamation: **This apply to the rest of the phases**. Once you reach the password copy it or write it down. You will need to "exit" the connection and connect to the exact same port but this time as bandit1, bandit2, bandit3... that will be changing depending on the level you are.
